- 请按照下面约定创建并使用账号，首次登录后**必须立即修改密码**。

## 账号规则
- 每个人的账号为**姓名全拼（小写）**，例如：张三 → `zhangsan`。

---

## 初始密码规则
- 初始密码格式：`xx_yy`
  - `xx` = 姓名全拼（与账号一致，均小写）
  - `yy` = 学号（学号按原样填写）
- 示例：账号 `zhangsan`，学号 `2022012345` → 初始密码：`zhangsan_2022012345`

---

## 首次登录后修改密码（强制操作）
请在首次登录后立即修改密码，步骤如下：

### 通过 macOS / Linux / WSL 登录（推荐）
1. 打开终端（Terminal）。
2. 使用 SSH 登录：

```bash
ssh zhangsan@10.176.64.113
```
（将 `zhangsan` 替换为你的账号名）

3. 系统会提示输入初始密码（`xx_yy` 格式），输入后回车。
4. 登录成功后，执行修改密码命令：

```bash
passwd
```

5. 系统会依次提示：输入当前密码、输入新密码、再次输入新密码以确认。

示例交互：
```
$ passwd
Changing password for user zhangsan.
Current password:   <-- 输入当前初始密码
New password:       <-- 输入你想设置的新密码（注意不可回显）
Retype new password:<-- 再次输入新密码
passwd: all authentication tokens updated successfully.
```

### 通过 Windows（PuTTY）登录
1. 启动 PuTTY，Host Name 填写：`10.176.64.113`，端口一般为 `22`，连接类型选 `SSH`。点 `Open`。
2. 出现终端窗口，系统提示 `login as:`，输入你的账号（如 `zhangsan`），回车。
3. 输入初始密码（`xx_yy` 格式），回车。
4. 登录成功后，在终端输入：

```bash
passwd
```

按照提示更改密码（输入当前密码 → 新密码 → 确认新密码）。
